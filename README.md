# Portfolio Website

Single-page portfolio website built with plain HTML, CSS, and JavaScript.

## Structure

- `index.html` - main one-page site with anchored sections
- `css/style.css` - visual system, responsive layout, and animations
- `js/script.js` - white tsParticles background, section reveal, nav scrollspy

## Notes

- Black background with white particles (`#tsparticles`)
- Includes placeholders for external proof links and evidence media
- No build tools required, open `index.html` in browser
